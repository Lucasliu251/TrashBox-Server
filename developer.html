基础信息

欢迎使用 TrashBox CS2 数据分析系统 API 接口文档。

  • 基础路径 (Base URL): https://trashbox.tech/api/v1
  • 数据交互格式: application/json (除文件上传外)
  • 接口鉴权: 微信小程序生产环境请在 Header 中携带 Token。

用户管理 (Users)

微信快捷登录

使用微信 wx.login 返回的临时 code 换取 OpenID，并检查用户是否已注册。

POST /users/login

请求参数 (Body JSON)

参数名类型必填描述
loginCodeString是微信临时登录凭证 (code)

新用户注册绑定

新用户绑定 Steam ID，支持自动解析包含乱码/自定义URL的输入内容。

POST /users/onboarding

请求参数 (Body JSON)

参数名类型必填描述
loginCodeString是微信临时登录凭证
steamIdString是用户输入的 Steam ID 或主页链接
authCodeString是完美平台鉴权码
matchCodeString是比赛专属特征码

获取个人资料

获取当前登录用户的详细档案资料。

GET /users/me

请求参数 (Query)

参数名类型必填描述
openidString是用户唯一标识 (开发环境暂以Query传入)

更新用户资料

用于修改自定义昵称或头像链接。

PUT /users/update

请求参数 (Body JSON)

参数名类型必填描述
openidString是微信用户的唯一标识
nicknameString否自定义昵称
avatarString否绑定的 头像URL

上传用户头像

处理小程序端用户自定义头像的上传请求，返回访问URL。

POST /users/avatar

Content-Type: multipart/form-data

请求参数

参数名类型必填描述
fileFile是图片文件实体 (JPG/PNG)

全局搜索用户

通过 Steam ID 精确查找或昵称模糊匹配玩家。优先返回已注册用户信息。

GET /users/search

请求参数 (Query)

参数名类型必填描述
qString是搜索关键词：昵称或SteamID (最少1字符)

玩家战绩 (Players & Stats)

获取玩家历史战绩与多维数据

获取指定 Steam 账号的近期游戏历史汇总、每日明细、多维数据评价与风格标签。

GET /players/{steam_id}/history

请求参数

参数名类型必填描述
steam_idString是玩家的 Steam 64位 ID (路径参数)
daysInteger否查询过去的天数，默认 30，最大 90 (Query参数)

返回示例 (节选)

{
  "steam_id": "76561198xxx",
  "nickname": "Lucas",
  "style_tag": "狠",
  "summary": { "avg_Rating": 1.25, "avg_kd": 1.15 },
  "history": [ ... ]
}

获取每日全服排行榜

获取某一日期的活跃玩家数据排名，离线聚类脚本也依赖此接口。

GET /rankings/daily

请求参数 (Query)

参数名类型必填描述
dateString是查询日期，格式 YYYY-MM-DD

社区文章 (Posts)

发布文章

允许注册用户发布富文本社区文章。

POST /posts/

请求参数 (Body JSON)

参数名类型必填描述
openidString是作者 UUID (OpenID)
titleString是文章标题
contentString是富文本 HTML 内容
tagString否分类标签 (默认"默认")

获取文章列表

首页信息流使用的分页获取文章列表接口，自动提取纯文本摘要与封面图。

GET /posts/

请求参数 (Query)

参数名类型必填描述
limitInteger否单页条数，默认 10
offsetInteger否偏移量，默认 0

获取文章详情

拉取文章正文及底部的评论列表，请求成功后文章 views 会自动加1。

GET /posts/{post_id}

请求参数

参数名类型必填描述
post_idInteger是文章的主键 ID (路径参数)

发表评论

对指定文章发表留言评论。

POST /posts/{post_id}/comment

请求参数 (Body JSON)

参数名类型必填描述
uuidString是评论者 OpenID
contentString是评论内容文本

上传文章图片

供富文本编辑器使用，上传图片后返回持久化链接。

POST /posts/upload

Content-Type: multipart/form-data

请求参数

参数名类型必填描述
fileFile是图片文件实体 (支持JPG/PNG/GIF/WEBP)

消息通知 (Notifications)

订阅微信通知

用户授权后，记录一次性订阅消息发送次数（剩余次数+1）。

POST /notifications/subscribe

请求参数 (Body JSON)

参数名类型必填描述
openidString是用户的 OpenID
template_idString是微信官方分配的模板ID

辅助与拓展 (Helpers)

Steam ID 智能解析 (独立辅助方法)

将用户输入的杂乱信息(自定义URL/纯ID/个人主页)归一化为标准的17位 SteamID64。

POST /bind

请求参数 (Body JSON)

参数名类型必填描述
steam_id_inputString是用户随意输入的 Steam 凭证信息

提交反应力测试成绩

记录用户产生的平均真实反应时间（TTK 数据源辅助）。

POST /reaction/submit

请求参数 (Body JSON)

参数名类型必填描述
steam_idString是玩家的 Steam ID
average_msInteger是平均反应时间 (毫秒)，如 195
